# VoteGuide

Your vote, made simple and accessible for everyone.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 18 + TypeScript
- Vite 6
- React Router v6
- Tailwind CSS v4
- shadcn/ui components

## Deploy on Vercel

1. Push to a Git repo
2. Import the repo on https://vercel.com
3. Vercel auto-detects Vite — no config needed
4. Add a rewrite rule for SPA routing (already handled by `vercel.json`)
